# Code of Conduct

## Our Pledge

We as contributors and maintainers pledge to make participation in this project a harassment-free experience for everyone, regardless of age, body size, disability, ethnicity, sex characteristics, gender identity and expression, level of experience, education, socio-economic status, nationality, personal appearance, race, religion, or sexual identity and orientation.

## Our Standards

**Positive behaviors:**
- Using welcoming and inclusive language
- Being respectful of differing viewpoints and experiences
- Gracefully accepting constructive criticism
- Focusing on what is best for the community
- Showing empathy towards other community members

**Unacceptable behaviors:**
- Trolling, insulting/derogatory comments, and personal or political attacks
- Public or private harassment
- Publishing others' private information without explicit permission
- Using this project's code for malicious purposes
- Any conduct which could reasonably be considered inappropriate

## Special Considerations for This Project

This repository contains intentionally malicious code for educational purposes. Contributors and users must:

1. **Never use this code maliciously** - The code is for education and authorized security testing only
2. **Respect the Hippocratic License** - All use must comply with human rights standards
3. **Promote responsible disclosure** - Report security concerns through proper channels
4. **Support the educational mission** - Help others learn about security, don't enable attacks

## Enforcement

Instances of abusive, harassing, or otherwise unacceptable behavior may be reported by contacting the project maintainer. All complaints will be reviewed and investigated promptly and fairly.

Project maintainers have the right and responsibility to remove, edit, or reject comments, commits, code, wiki edits, issues, and other contributions that are not aligned with this Code of Conduct.
